# NFT Gallery Project

Building an NFT Gallery which can fetch NFTs with their metadata, display images, descriptions & ids based on wallet address, collection address using Alchemy NFT API, Polygon, next.js and Tailwind CSS.

## Built With

- Next.js
- Alchemy NFT API
- Tailwind CSS
- Polygon Mainnet
